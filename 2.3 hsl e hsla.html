Cores

Existem várias formas de definirmos cores utilizando o CCS. Elas são:

  • cores pré-definidas
  • palavras-chave "currentcolor"
  • Hexadecimal
  • Hexadecimal com transparência
  • RGB
  • RGBA
  • HSL
  • HSLA

HSL

hsl( hue, saturation, lightness)
  • hue (matriz): é o grau na roda de cores (de 0 a 360 graus)
    • 0 ou 360 é vermelho
    • 120 é verde
    • 240 é azul
  • saturation (saturação): é a intensidade da cor (de 0% a 100%)
    • 0% é cinza
    • 100% é a cor pura
  • lightness (luminosidade): é a quantidade de branco ou preto na cor (de 0% a 100%)
    • 0% é preto
    • 100% é branco
