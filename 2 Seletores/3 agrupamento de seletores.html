Agrupamento de seletores

Podemos aplicar as mesmas reguras Css para Varios seletores diferentes. Para isso, podemos separar os seletorews utilizando uma virgula(,).

Assim, não podemos ficar copiando e coloando os mesmos códigos para todos, nós podemos fazer uma só bloco de CSS e reutilizar as mesmas regras!
